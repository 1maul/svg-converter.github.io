SVG File Modifier

This tool modifies SVG files to match the structure of the provided template. Upload an SVG file or paste its content below, then click "Modify SVG" to transform it.

The modified SVG will:

  • Use the same XML declaration and namespace
  • Apply the same viewBox dimensions (0 0 380 600)
  • Include the same style definition with color classes
  • Wrap all paths in a <g id="Layer_1"> element
  • Preserve all original path data but apply the selected color class
